Dimensi Rohani

Salib Emas

Melambangkan pengorbanan Kristus dan kemuliaan Tuhan yang menjadi pusat dari komunitas Perokris. Warna emas memancarkan kemuliaan yang abadi dan tak lekang oleh waktu.

  • Pusat Komunitas
  • Kemuliaan Tuhan
  • Pengorbanan Suci
  • Keabadian Iman

Alkitab Terbuka

Melambangkan firman Tuhan sebagai dasar atau pondasi dalam bekerja dan berperilaku di lingkungan kantor. Terbuka berarti siap menerima hikmat setiap hari.

  • Pondasi Hikmat
  • Keterbukaan Hati
  • Dasar Perilaku
  • Panduan Aktivitas

Burung Merpati

Melambangkan Roh Kudus, perdamaian, dan kasih. Posisi terbang menggambarkan semangat untuk terus maju, bertransformasi, dan melayani dengan tulus.

  • Roh Kudus
  • Kasih & Damai
  • Transformasi Diri
  • Semangat Melayani
Filosofi Kedalaman

Salib sebagai Batu Penjuru

Dalam desain ini, Salib diletakkan di tengah dan menjadi titik pusat (center). Sebuah batu penjuru tidak harus menjadi bagian terbesar, tapi dia adalah penentu arah dan kekuatan seluruh konstruksi.

Salib berfungsi sebagai Navigasi. Ukurannya yang proporsional melambangkan bahwa iman Kristiani bukan untuk dipamerkan secara sombong, tapi untuk dirasakan dampaknya secara nyata.

  • Navigasi Iman
  • Titik Pusat
  • Penentu Arah
  • Kekuatan Nyata

Bingkai Kelopak / Api Biru

Merepresentasikan lidah-lidah api yang melambangkan turunnya Roh Kudus untuk memberi keberanian dan hikmat dalam bekerja. Api biru melambangkan suhu tertinggi namun tetap tenang dan stabil.

  • ✓ Keberanian Bekerja
  • ✓ Profesionalisme Tinggi
  • ✓ Berkepala Dingin
  • ✓ Stabilitas Rohani
